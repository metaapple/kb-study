# 자바(Java) 기본 학습 가이드

## 📚 개요
이 프로젝트는 자바(Java) 프로그래밍의 기초를 학습하기 위한 지식 기반입니다. 초보자부터 중급자까지 자바의 핵심 개념을 이해할 수 있도록 구성되었습니다.

## 🎯 학습 목표
- 자바의 기본 문법 이해
- 객체 지향 프로그래밍(OOP) 개념 학습
- 자바 핵심 라이브러리 활용
- 프로젝트 기반의 실전 학습

## 📖 주요 학습 주제

### 1. 자바 기초
- **자바란?** - 플랫폼 독립적인 객체 지향 프로그래밍 언어
- **JDK, JRE, JVM** - 자바 개발 환경 이해
- **첫 프로그램** - "Hello, World!" 작성
- **변수와 자료형** - int, String, boolean 등

### 2. 연산자와 제어문
- **산술 연산자** - +, -, *, /, %
- **비교 연산자** - ==, !=, <, >
- **논리 연산자** - &&, ||, !
- **조건문** - if, else if, else, switch
- **반복문** - for, while, do-while

### 3. 배열과 컬렉션
- **배열** - 같은 타입의 여러 데이터 저장
- **List** - ArrayList, LinkedList
- **Set** - HashSet, TreeSet
- **Map** - HashMap, TreeMap

### 4. 객체 지향 프로그래밍(OOP)
- **클래스와 객체** - 설계도와 실제 제품
- **생성자(Constructor)** - 객체 초기화
- **메서드** - 객체의 동작 정의
- **캡슐화** - private, public, protected
- **상속(Inheritance)** - 코드 재사용
- **다형성(Polymorphism)** - 같은 이름, 다른 동작
- **추상화(Abstraction)** - abstract 클래스와 interface

### 5. 예외 처리
- **try-catch-finally** - 예외 처리
- **throw와 throws** - 예외 던지기
- **Custom Exception** - 사용자 정의 예외

### 6. 입출력(I/O)
- **Scanner** - 사용자 입력 받기
- **System.out** - 출력하기
- **파일 입출력** - FileReader, FileWriter
